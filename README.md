# Geo Base

Geo Baseは、国別・時系列データから、世界地図上の勢力変化を連続アニメーションとして描画するRemotionベースの縦型動画生成エンジンです。

## 現在の動画

### ドイツと日本はどこまで拡大した？ 第二次世界大戦・占領地勢力図

1939年9月から1945年9月まで、第二次世界大戦中の主要勢力による実効支配地域の拡大・縮小を、月次の連続アニメーションとして表示します。

- ドイツ、日本、イタリア、ソ連、イギリス、アメリカ、フランスを比較
- 現代の国境へ換算した概算支配割合を複数色で表示
- 月次キーフレーム間を連続補間し、色の境界を滑らかに変化
- 開戦時点から新たに獲得した推定陸地割合をランキング化
- フランス降伏、独ソ戦、太平洋戦争、スターリングラード、ノルマンディー上陸などを長めのイベント表示で解説
- 日本中心の世界地図を採用
- 日本・東アジア・東南アジアの拡大図を南太平洋上へ配置
- スマートフォンで読みやすい大型文字と余白を使用
- 最終画面に最大拡大時期、結果、議論用コメントテーマを表示

## 歴史データについて

付属データは、米陸軍戦史、Library of Congress、公開されている歴史地図などを照合し、動画制作用に再構成した概算データです。

厳密な月別前線や法的主権を示すデータではありません。現在の国境ポリゴンごとに、各勢力の実効支配割合を推定して表示しています。同盟国、占領地域、植民地、委任統治領、軍政地域は同じ意味ではないため、動画内では「実効支配・勢力圏の概算」として扱います。

公開前には、重要な転換点と地域について一次資料・信頼できる歴史資料との再確認が必要です。

## 主な機能

- `YYYY-MM`形式の月次データを検証
- ISO 3166-1 alpha-3国コードと数値コードを検証
- 同一期間・同一地域の勢力値を100%へ正規化
- 国の内部を複数勢力の比率で表示
- 陸地面積の重みを利用した勢力ランキング
- 初期期間との差分から新規獲得地域を集計
- 月次スナップショット間を連続補間
- 日本中心の世界地図を生成
- 南太平洋上に日本・太平洋戦線の拡大図を表示
- 順位とランキングバーを連続的に入れ替え
- 重要イベントを複数月にまたがって表示
- 1080×1920 / H.264 MP4を生成

## 出力ファイル名

```text
GeoBase_<動画タイトル>_<開始期間>-<終了期間>.mp4
```

現在の設定では次のファイル名になります。

```text
GeoBase_ドイツと日本はどこまで拡大した？_第二次世界大戦・占領地勢力図_1939-09-1945-09.mp4
```

ファイル名として使用できない記号は自動的に除去され、空白はアンダースコアへ変換されます。

## GitHub Actionsで動画を生成する

### マージ後の自動生成

Pull Requestを`main`へマージすると、`main`へのpushを検知してGitHub Actionsが自動的に起動します。

1. 月次データ生成
2. データ検証・集計
3. TypeScript型検査
4. 日本語フォントのインストール
5. Remotionブラウザの準備
6. MP4レンダリング
7. Artifactへのアップロード

実行完了後、対象のWorkflow Runを開き、`Artifacts`から`GeoBase-video-実行番号`をダウンロードしてください。

### 手動で再生成する

1. GitHubリポジトリ上部の`Actions`を開く
2. `Render Geo Base Video`を選択する
3. `Run workflow`を押す
4. 実行完了後、`Artifacts`から動画をダウンロードする

Pull Request作成時は検証のみを実行します。Artifactの保存期間は14日です。

## データ差し替え

編集対象は次の4ファイルです。

- `public/data/sample.csv`：月次の実効支配割合と陸地面積の重み
- `public/data/entities.json`：勢力名、色、表示記号、ランキング対象
- `public/data/config.json`：タイトル、期間、速度、拡大図位置、コメントテーマなど
- `public/data/events.csv`：重要月に表示する歴史イベント

### CSV必須列

```csv
period,country_code,country_name,numeric_code,entity,value,weight
1942-06,CHN,China,156,Japan,38.0,9597
1942-06,CHN,China,156,Other,62.0,9597
```

- `period`は`YYYY-MM`形式です。
- `country_code`はISO 3166-1 alpha-3を使用します。
- `numeric_code`は世界地図との対応に使うISO数値コードです。
- `value`は同一期間・同一地域で100%へ正規化されます。
- `weight`はランキング計算に使用する概算陸地面積です。
- データのない地域はグレー表示されます。

## 日本・太平洋拡大図の配置ルール

拡大図は、オーストラリアとチリの間にある南太平洋上の空き領域へ配置します。

- 世界地図の下側
- オーストラリア、ニュージーランド、北米、南米と重ならない位置
- 下部イベントバナーと干渉しない高さ
- `config.json`の`insetLeft`と`insetTop`で調整可能

現在の基準値は次のとおりです。

```json
{
  "insetLeft": 560,
  "insetTop": 390
}
```

## 現在の制約

- 付属データは動画制作用に再構成した概算値です。
- 現代の国境を利用しているため、当時の国境線や前線を完全には再現しません。
- 一つの現代国家内に複数の支配地域があった場合は、割合による色分けで近似します。
- 海戦、制海権、空域、補給線は地図の面積ランキングへ含めません。
- 音声、BGM、自動ナレーションは未実装です。
